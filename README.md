# app-online-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常为离线或在线环境下可直接打开的说明文档、交互工具或参考指南。

## 项目简介

`app-online-guide` 是一个静态页面集合仓库。每个 HTML 文件均可独立运行，不依赖后端服务或特定域名。适用于存放各类技术说明、操作指引、工具页面或演示内容。

## 目录结构

```
.
├── README.md          # 本文件
├── index.html         # 可选的主索引页
├── pages/             # 存放各个独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 公共资源文件（CSS、JS、图片等）
    └── ...
```

- `pages/`：主要页面目录，每个 `.html` 文件为一个独立页面。
- `assets/`：存放多个页面共用的样式、脚本或图像文件。

## 页面归档说明

- 每个 HTML 页面应尽量自包含，减少对外部资源的依赖。
- 如需引用共享资源，请放置于 `assets/` 目录下，并在页面中通过相对路径引用。
- 页面命名建议使用英文小写字母与连字符，如 `quick-start-guide.html`。
- 新页面添加后，可在 `index.html` 中添加链接以便浏览。

## 维护说明

- 本仓库接受合理的页面新增与更新。
- 提交前请确保页面在主流浏览器中可正常打开。
- 不建议在页面中包含任何具体域名、网址或诱导性内容。
- 资源文件请保持精简，避免引入过多第三方库。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 目录下的任意 `.html` 文件。
3. 或通过 GitHub Pages 等静态托管服务部署后访问。
